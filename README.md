Twit project
============

Installation
------------
1. Copy `app/config/app.sample.json` to `app/config/app.json` and update configuration details to that file accordingly.

2. Install dependencies

        bower install
        npm install

3. Start the server

        npm start
    

Open `http://localhost:3000` in the browser
